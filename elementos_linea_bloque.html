Elementos HTML

Elementos en bloque

Son elementos que en su configuracion original ocupan todo el ancho de la pantalla y que desplazan a otros elementos para ocupar un espacio en la pantalla

Ejemplos de elementos en bloque son:

  • Elementos h o headers (h1,h2...h6)
  • Elemento p
  • Elemento li
  • Elemento table
  • Elemento div
  • Etc...

Elementos en linea

Son elementos que se ajustan al contenido de la pagina y que no ocupan todo el espacio de pantalla

Ejemplos de elementos en linea son:

  • Elemento a
  • Elemento img
  • Elemento strong
  • Elemento b
  • Etc...
